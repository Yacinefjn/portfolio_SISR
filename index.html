ACKnowledge
▶ Stage de Première Année · 02/06/2025 – 27/06/2025
Cybersécurité

Type : Intégrateur de solutions de cybersécurité

Objectif : Étude et mise en pratique du modèle ZeroTrust

Netskope
Plateforme SASE intégrant CASB, SWG et ZTNA pour sécuriser l'accès aux applications cloud dans une approche Zero Trust.
BeyondTrust
Solution PAM pour sécuriser et gérer les accès privilégiés avec le principe du moindre privilège.
Illumio
Plateforme de microsegmentation Zero Trust pour contenir les cybermenaces et empêcher la propagation latérale.
Réalisation Technique — Maquette ZeroTrust sous VMware
  • Configuration d'un client Windows avec 3 agents (Netskope, BeyondTrust, Illumio)
  • Déploiement de 2 serveurs Linux isolés
  • Configuration des communications client↔serveurs avec isolation inter-serveurs
  • Application des politiques de segmentation Zero Trust
  • Gestion de politiques SASI (Proxy + Firewall)
  • Gestion granulaire des politiques des utilisateurs
